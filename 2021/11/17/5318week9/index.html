Clustering

Clustering Applications:Image Discretization

Reassign colour value to k distinct colours Cluster pixels using colour difference,not spatial data

Measuring Similarity/Distance

Measure the distance(d) between two points(a,b)
The distance d(a,b) indicate us how similar the objects a and b are.
· If d(a,b) is small, the objects a and b are similar
· If d(a,b) is large, the objects a and b are dissimilar

测量两点(a,b)的距离(d)

距离d(a,b)表示物体a和b有多相似。

·如果d(a,b)较小，则a和b的对象相似

·如果d(a,b)是大的，则对象a和对象b是不相同的

The goal of clustering is”putting nearby points (small distance)into the group”

聚类的目标是“将附近的点(小距离)放入组中”。

Clustering Algorithms:Partitioning Clustering

  • 聚类算法:分区聚类

分区聚类分区聚类是一种聚类方法，需要分析人员指定要生成的聚类数量。

最著名的划分聚类算法家族是k-均值聚类。

K-means Clustering
·Number of clusters Knust be specified
·Each cluster is associated with a centroid (center point)
·Each point is assigned to the cluster with the closest centroid

k - means聚类

·必须指定集群的数量

·每个集群与一个质心(中心点)相关联

·每个点被分配到最接近质心的聚类中

Procedure

  1. Select a number of groups to use and randomly initialize their respective center points.
  • The center points are vectors of the same length as each data point vector and are the “X’s”in the graphic.
  1. Each data point is classified by computing the distance between that point and each group center, and then classifying the point to be in the group whose center is closest toit.
  2. Based on these classified points, recompute the group center by taking the mean of all the vectors in the group.
  3. Repeat these steps for a set number of iterations or randomly initialize the group centers, and then selectthe run that looks like it provided the best results.

\1. 选择一些组来使用并随机初始化它们各自的中心点。

*中心点是与每个数据点向量长度相同的向量，是图形中的“X”。

\2. 每个数据点的分类是通过计算该数据点与组中心之间的距离，然后将该数据点分类为最接近该数据点的组内点。

\3. 在这些分类点的基础上，取群中所有向量的均值，重新计算群中心。

\4. 对一组迭代或随机初始化组中心重复这些步骤，然后选择看起来能提供最佳结果的运行。

K-means convergence(Stopping) Criterion
8 no(or minimum) re-assignments of data points to different clusters,or
·no lor minimum) change of centroids,or
·minimum decrease in the sum of squared error(SSE)

k - means收敛准则(停止)

没有(或最少)将数据点重新分配到不同的集群，或

·无最小值)质心变化，或

·误差平方和(SSE)最小降幅

Handling Empty Clusters

K-means Clustering-Handling Empty Cluster Issue K-means can yield empty clusters Several strategies
· Choose the point that is farthest away from any current centroid. If nothing else, this eliminates the point that currently contributes most to the total squared error.
A K-means++approach could be used as well.
· Choose the replacement centroid at random from the cluster that has the highest SSE.
· This will typically split the cluster and reduce the overall SSE of the clustering. If there are several empty clusters, the above can be repeated several times.

K-means聚类-处理空聚类问题

·选择离任何电流质心最远的点。 如果没有别的，这就消除了目前对总平方误差贡献最大的点。

*也可以使用k -means++方法。

·从SSE最高的聚类中随机选取替换质心*

·这通常会分裂集群，降低集群的总体SSE。 如果有几个空集群，上述操作可以重复几次。

Outliers

When the squared error criterion is used, outliers can unduly influence the clusters that are found.
In particularwhen outliers are present, the resulting cluster centroids
(prototypes) are typically not as representative as they otherwise would be and thus, the SSE will be higher.

If we use approaches that remove outliers before clustering, we avoid clustering points that will not cluster well.
Alternatively, outliers can also be identified in a postprocessing step.

Updating Centers Incrementally

Instead of updating cluster centroids after all points have been assigned to a cluster, the centroids can be updated incrementally, after each assignment of a point to a cluster.

·Each assignment updates zero or two centroids
·More expensive(-)
·Introduces an order dependency(-)
·Never get an empty cluster
·Can use”weights”to change the impact

Bisecting K-means

A straightforward extension of the basic K-means algorithm that is based on a simple idea: to obtain K clusters, split the set of all points into two clusters, select one of these clusters to split, and so on, until K clusters have been produced.

>bisecting K-means is less susceptible to initialization problems.

K-means 局限

K-means and its variations have a number of limitations with respect to finding different types of clusters. In particular,K-means has difficulty detecting the
“natural”clusters, when clusters have non-spherical shapes or widely different sizes or densities.

优点和缺点

K-means is simple and can be used for a wide variety of data types.
It is also quite efficient, even though multiple runs are often performed. Some variants, including bisecting K-means, are even more efficient, and are less susceptible to initialization problems.
K-means is not suitable for all types of data, however. It cannot handle non-
globular clusters or clusters of different sizes and densities, although it can typically find pure subclusters if a large enough number of clusters is specified.
K-means also has trouble clustering data that contains outliers. Outlier detection and removal can help significantly in such situations.

K-means很简单，可以用于各种数据类型。

它也非常高效，即使经常执行多次运行。 有些变体(包括平分K-means)甚至更有效，而且不容易出现初始化问题。

然而，K-means并不适用于所有类型的数据。 它不能处理非

球状星团或大小和密度不同的星团，尽管如果指定足够多的星团，它通常可以找到纯亚星团。

K-means在聚类包含离群值的数据时也存在问题。 在这种情况下，离群值检测和删除可以提供很大帮助。

Model base

Gaussian Mixture Models

GMM with Expectation-Maximization (EM) 再看看

Gaussian Mixture Models(EM Clustering)
·Assume that the data generating process is a mixture of Multivariate Normals

Gaussian Mixture Models (EM Clustering)
·Assume that the data generating process is a mixture of Multivariate Normals
·Assume Tk is the weight of the k-th Multivariate Normals in the mixture
·Assume that all observations from cluster k are drawn randomly from a MUN(uk,Zk) distribution
Multivariate Normals

高斯混合模型(EM聚类)

·假设数据生成过程是多元法线的混合

·假设Tk为混合物中第k个多元法线的权重

·假设集群k的所有观测数据都是从MUN(uk,Zk)分布中随机抽取的

多元法线

·We don’t know what the mixture components look like ahead of time
·We’ll use an iterative algorithm that feels kind of like k-means
·Start by fixing k at some value
Calculate the probability that each point belongs to each distribution
Use these probabilities to compute a new estimate for the parameters

·我们事先不知道混合成分是什么样子的

·我们将使用一种类似于k-means的迭代算法

·首先将k固定在某个值上

计算每个点属于每个分布的概率

使用这些概率来计算参数的新估计

Hierarchical Clustering

Hierarchical Clustering
·Produces a set of nested clusters organized as a hierarchical tree
·Can be visualized as a dendrogram
·A tree like diagram that records the sequences of merges or splits

Agglomerative
·Start with the points as individual clusters
·At each step,merge the closest pair of clusters until only one cluster(or k clusters)left

烧结的

·从单个集群的点开始

·每一步合并最接近的一对簇，直到只剩下一个簇(或k个簇)

Divisive
·Start with ope,all-inclusive cluster
·At each step,split a cluster until each cluster contains an individual point(or there are k clusters)

分裂的

·从ope开始，全包集群

·每一步拆分一个聚类，直到每个聚类包含一个单独的点(或有k个聚类)

Strengths and Weaknesses

More generally, such algorithms are typically used because the underlying application,e.g, creation of a taxonomy, requires a hierarchy.
Also, some studies have suggested that these algorithms can produce better-
quality clusters.
However, agglomerative hierarchical clustering algorithms are expensive in terms of their computational and storage requirements.
The fact that all merges are final can also cause trouble for noisy, high-dimensional data, such as document data.
In turn, these two problems can be addressed to some degree by first partially clustering the data using another technique, such as K-means.

更一般地说，通常会使用这种算法，因为底层应用程序e。 分类法的创建需要层次结构。

此外，一些研究表明，这些算法可以产生更好的-

质量集群。

然而，聚类分层聚类算法在计算和存储方面的开销很大。

事实上，所有的合并都是final的，这也会给文档数据等嘈杂的高维数据带来麻烦。

反过来，通过使用另一种技术(如K-means)首先部分聚类数据，可以在一定程度上解决这两个问题。

花费多 对noise处理弱
